
## Description

A telegram bot that gets movies based on user input , currently in progress.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Running the bot

```bash
# step 1
open telegram

# step 2
search for bot name (kennyFilmBot) on telegram

# step 3
enter commands
```
